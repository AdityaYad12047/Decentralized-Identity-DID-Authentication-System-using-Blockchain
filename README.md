# 🌐 PROJECT OVERVIEW

The **Decentralized Identity (DID) Authentication System** is a production-oriented blockchain security platform engineered to modernize digital identity verification using decentralized authentication protocols and AI-driven fraud intelligence.

Built under **IBM Internship Guidance**, the system eliminates reliance on centralized identity providers by leveraging:

<table>
<tr>
<td width="50%">

## 🔗 Blockchain Security Layer

* Ethereum Smart Contracts
* Wallet-based authentication
* Decentralized identity ownership
* Immutable DID verification
* Cryptographic trust validation

</td>

<td width="50%">

## 🧠 AI Fraud Intelligence Layer

* Random Forest fraud detection
* Behavioral anomaly analysis
* Authentication risk scoring
* Suspicious activity detection
* Real-time security intelligence

</td>
</tr>
</table>

# ⚡ INSTALLATION

## Clone Repository

```bash
git clone https://github.com/AdityaYad12047/Decentralized-Identity-DID-Authentication-System-using-Blockchain.git
```

## Backend Setup

```bash
cd backend
npm install
npm start
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## ML Service

```bash
cd ml
pip install -r requirements.txt
python app.py
```
